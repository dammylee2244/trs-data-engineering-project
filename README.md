# TRS Data Engineering Project

## Overview
End-to-end PySpark ETL pipeline for retirement member analytics.

## Technologies
- Python
- PySpark
- Apache Spark
- Git/GitHub
- Parquet
- SQL

## Features
- CSV ingestion
- Data transformations
- Aggregations
- Analytics-ready outputs
- Parquet storage generation

## Architecture
CSV Files → PySpark → Transformations → Parquet Output
